# Personal website

This project is my personal website, containing some information about me and links to some of my most recent projects. I built everything from scratch using React Vite and Spline for the first time:
- I designed the website, choosing the theme and colors,
- I created and merged the components,
- I have learned to use spline,
- I have made everything responsive,
- Finally, I deployed the website with Vercel.
